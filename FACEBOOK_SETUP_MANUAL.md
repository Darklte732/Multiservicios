## STEP 1 — Rename the page for SEO (5 min)

The current name "BaezMultiservicios" misses every important search keyword. People in El Seibo search "electricista el seibo", "electricista cerca de mi", "neno electricista". Your name should contain those words.

### How to rename
1. Go to your page (you're there now)
2. Click the **Edit** button next to "Professional Dashboard"
3. Find the "Page name" field
4. Change to: **`MultiServicios El Seibo - Electricista Neno Báez`**
5. Save

**Note:** Facebook reviews name changes within 1–3 days. Don't change it again for 60 days after approval. Choose carefully.

### If you want a shorter name option
- `MultiServicios El Seibo` (cleaner, less keyword stuffing)
- `Electricista Neno - El Seibo` (most direct)

**Recommendation:** First option (longer, keyword-rich). Facebook search ranking rewards keyword match in the page name.

### Username (handle)
At the same edit screen, set username to: **`@MultiServiciosElSeibo`**
This makes your URL: facebook.com/MultiServiciosElSeibo (much easier to share)

## STEP 12 — Domain Verification + Pixel (do this with your developer)

This unlocks the ad pixel that tracks conversions from Facebook → website.

1. Meta Business Suite → Settings → Brand safety → Domains
2. Add `multiservicios.app`
3. Verify via DNS TXT record (give the value to whoever manages your domain)
4. Once verified, install Meta Pixel on the website (Next.js — add the Pixel snippet to `app/layout.tsx`)

**Pixel snippet location:** I noted in the strategy doc that this is critical for ads ROI. The Pixel ID is in Meta Business Suite → Events Manager → Pixels.
